## Deskripsi

`ClickableWrittenExpressionQuiz` adalah komponen kuis yang dirancang untuk menguji pemahaman tata bahasa dan struktur kalimat. Dalam kuis ini, pengguna disajikan beberapa kalimat, di mana setiap kalimat memiliki satu kata yang salah secara gramatikal.

Pengguna harus mengidentifikasi dan mengklik kata yang salah dari beberapa pilihan yang telah ditentukan. Komponen ini secara otomatis akan mengubah kata-kata pilihan dalam kalimat menjadi tombol yang bisa diklik, sementara sisa kalimatnya tetap sebagai teks biasa.

## Contoh Implementasi

    correctAnswerKey: "C",
    feedback: "Kata 'مُجْتَهِدٌ' adalah khabar kāna (predikat untuk kāna), sehingga ia harus berada dalam keadaan nashab (mansub). Bentuk i'rab yang benar adalah 'مُجْتَهِدًا'."

    feedback: "Kata 'حَاضِرَانِ' berfungsi sebagai khabar laysa dan harus dalam keadaan nashab. Bentuk nashab untuk mutsanna (dual) adalah dengan 'ي', sehingga bentuk yang benar adalah 'حَاضِرَيْنِ'."

## Properti (Props)

| Properti | Tipe | Wajib | Default | Deskripsi |
| :--- | :--- | :--- | :--- | :--- |
| `title` | `String` | Tidak | - | Judul yang akan ditampilkan di bagian atas kuis. |
| `questions` | `Array` | Ya | - | Sebuah array berisi objek-objek pertanyaan untuk kuis. |

---

## Struktur Data `questions`

Setiap elemen dalam array `questions` adalah sebuah objek dengan struktur berikut:

```js
{
  // ID unik untuk setiap pertanyaan (Wajib)
  id: 'kana1',

  // Kalimat lengkap yang mengandung kesalahan (Wajib)
  sentence: "كَانَ الطَّالِبُ مُجْتَهِدٌ.",

  // Array objek yang berisi pilihan kata yang bisa diklik (Wajib)
  options: [
    { key: 'A', text: "كَانَ" },
    { key: "B", text: "الطَّالِبُ" },
    { key: "C", text: "مُجْتَهِدٌ" },
  ],

  // Kunci dari opsi yang merupakan jawaban benar (Wajib)
  correctAnswerKey: "C",

  // Penjelasan yang muncul setelah soal dijawab (Opsional)
  feedback: "Penjelasan mengapa jawaban tersebut salah dan apa bentuk yang benar."
}
```

### Penjelasan Struktur Data

- **`id`**: String (Wajib). ID unik untuk setiap soal.
- **`sentence`**: String (Wajib). Kalimat lengkap yang akan ditampilkan. Komponen akan secara otomatis mencari kata-kata dari `options` di dalam string ini.
- **`options`**: Array of Objects (Wajib). Setiap objek merepresentasikan satu pilihan yang bisa diklik.
  - `key`: String unik untuk pilihan tersebut (misal: 'A', 'B').
  - `text`: Teks kata yang akan dicari dalam `sentence` dan dijadikan tombol.
- **`correctAnswerKey`**: String (Wajib). `key` dari pilihan yang benar.
- **`feedback`**: String (Opsional). Umpan balik yang ditampilkan setelah pengguna menjawab.

## Fitur Unggulan

- **Render Cerdas**: Komponen secara otomatis memecah kalimat dan mengubah kata-kata dalam `options` menjadi tombol interaktif.
- **Fokus Pengguna**: Dengan hanya menyorot kata-kata yang relevan, kuis ini membantu pengguna fokus pada analisis bagian kalimat yang penting.
- **Umpan Balik Langsung**: Pengguna langsung mengetahui apakah jawabannya benar atau salah untuk setiap soal.
- **Konteks Utuh**: Meskipun hanya beberapa kata yang interaktif, kalimat tetap ditampilkan secara utuh, menjaga konteks gramatikalnya.
