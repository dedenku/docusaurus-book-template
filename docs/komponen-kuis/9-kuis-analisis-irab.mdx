## Deskripsi

`IrabAnalysisQuiz` adalah komponen kuis tingkat lanjut yang dirancang untuk menguji pemahaman mendalam tentang *i'rab* (analisis gramatikal) dalam tata bahasa Arab. Pengguna ditantang untuk menganalisis kata-kata tertentu dalam sebuah kalimat.

Untuk setiap kata yang dipilih, pengguna harus menentukan tiga aspek *i'rab*-nya: **Kedudukan (الموقع)**, **Keadaan (الحالة)**, dan **Alamat (العلامة)** melalui serangkaian menu dropdown. Komponen ini menggunakan format slider, menyajikan satu kalimat (soal) pada satu waktu untuk menjaga fokus.

## Properti (Props)

| Properti | Tipe | Wajib | Default | Deskripsi |
| :--- | :--- | :--- | :--- | :--- |
| `title` | `String` | Tidak | - | Judul yang akan ditampilkan di bagian atas kuis. |
| `questions` | `Array` | Ya | - | Sebuah array berisi objek-objek pertanyaan untuk kuis. |

---

## Struktur Data `questions`

Setiap elemen dalam array `questions` adalah sebuah objek soal yang memiliki struktur sebagai berikut:

```js
{
  // ID unik untuk soal (Wajib)
  id: 'q_panjang_1',

  // Kalimat lengkap yang akan ditampilkan (Wajib)
  sentence: "...",

  // Array berisi kata-kata dari kalimat di atas yang akan dianalisis (Wajib)
  words: [
    {
      // Kunci unik untuk kata ini (Wajib)
      key: 'qp1w1',

      // Teks kata yang akan dicari dalam `sentence` dan dijadikan tombol (Wajib)
      text: "الْمُهَنْدِسُوْنَ",

      // Objek berisi pilihan dropdown untuk setiap aspek i'rab (Wajib)
      options: {
        kedudukan: ['Fa\'il', 'Maf\'ul Bih', 'Mubtada'],
        keadaan: ['Marfu\'', 'Mansub', 'Majrur'],
        alamat: ['Dhommah', 'Waw', 'Alif'],
      },

      // Objek berisi jawaban yang benar untuk setiap aspek (Wajib)
      correctAnswer: {
        kedudukan: 'Fa\'il',
        keadaan: 'Marfu\'',
        alamat: 'Waw',
      }
    },
    // ...objek kata lainnya
  ]
}
```

## Fitur Unggulan

- **Analisis Detail**: Memungkinkan pengujian pada tiga tingkat *i'rab* sekaligus untuk satu kata.
- **Format Slider**: Kuis disajikan soal per soal, membantu pengguna fokus pada satu kalimat kompleks pada satu waktu.
- **Navigasi Terpandu**: Pengguna harus menyelesaikan semua analisis kata dalam satu soal sebelum bisa lanjut ke soal berikutnya.
- **Umpan Balik per Kata**: Pengguna dapat memeriksa jawabannya untuk setiap kata secara individual sebelum menyelesaikan seluruh soal.
